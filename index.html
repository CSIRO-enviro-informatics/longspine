LongSpine DB

This small, static website is a write-up of the models and data assembled by the Longitudinal Spine of Government Functions project (LongSpine) which ran in the 2018/2019 Financial Year as part of CSIRO's Data61's Platforms for Open Data (PfOD) series of projects.

See the About Page for more information on the project.

Figure 1: The overarching concepts of the Longitudinal Spine

What you can do here:

  • Learn about the motivating design choices of LongSpine - Principles Page
  • See all the models - Models Page
    • The overarching and component models used in this project are listed here as are links to their definitional homes
  • Learn about the individual datasets - Data Page
  • Understand the mechanics of this cache of the LongSpine spine - Cache Page
  • Query the cache - SPARQL Query Endpoint Page
    • We maintains here a cache of all the models and data within an RDF triplestore which you can query using the SPARQL query language
    • Yuo can get raw results or download results as CSV files
    • See the Examples Page for ready-to-go queries
  • See example queries and other data use scenarios - Examples Page
  • Learn more about the LongSpine project overall - About Page
